#### Hangman

Language: Python

This program uses the Cozmo SDK and command line input to have Anki's Cozmo robot host a game of hangman

##### Requirements

* Hardware
  * Cozmo robot
* Software
  * Python (Computer)
  * Cozmo app running in SDK mode (Phone)
  
##### Executing

* Execute Python file (.py files) at the command line:
  * Py hangman.py
